# Flappy Bird (Sprint 2)

Игра на движке LibGDX, созданная в рамках учебного проекта.

## Особенности
- Динамическая сложность (скорость увеличивается со временем)
- Система рекордов (High Score)
- Анимация взмахов крыльев птицы
- Плавная прокрутка фона

## Как запустить
1. Склонируйте репозиторий.
2. Откройте проект в Android Studio или IntelliJ IDEA.
3. Запустите модуль `lwjgl3` для игры на ПК.
